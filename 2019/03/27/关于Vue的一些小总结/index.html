定义方法

vue如果需要定义一些方法处理逻辑，则需要在methods中定义,调用时与上面变量的调用一样this.functionA()调用

生命周期

vue实例在整个生命周期中有四个主要阶段，共8个生命周期钩子，当vue实例的生命周期到某个阶段时，会自动触发钩子中申明的方法，可以查看官方指南

钩子名 说明
beforeCreate 实例创建前
created 实例创建后
beforeMounte 实例装载前
mounted 实例装载后
beforeUpdate 数据更新前
updated 数据更新后
beforeDestroy 实例销毁前
destroyed 实例销毁后

计算属性

监听属性
